## ğŸš€ Getting Started

Ready to modernize farming? Follow these steps to set up the project locally.

### Prerequisites
*   **Node.js** (v16+)
*   **npm** or **yarn**

### 1. Clone the Repository
```bash
git clone https://github.com/DEVELOPER-DEEVEN/AGRI-OS.git
cd AGRI-OS
```

### 2. Install & Run Backend
```bash
cd backend
npm install
# Create a .env file if needed (see .env.example)
node server.js
```
> Server runs on `http://localhost:5000`

### 3. Install & Run Frontend
```bash
# Open a new terminal
cd frontend
npm install
npm run dev
```
> App runs on `http://localhost:5173`
